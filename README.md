# jogo-de-dados
 
Criei um aplicativo de dados, onde você poderá jogar os dados apertando um simples botão!

O QUE APRENDI NESSE PROJETO:

- Criar um app com funcionalidade
- Usar IBActions e IBOutlets.
- Gerar números aleatórios.
- Auto Layout
